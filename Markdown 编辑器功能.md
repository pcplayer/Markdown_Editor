# Markdown 编辑器功能

## 基本功能
1. **打开文件**：打开一个 .md 文件，编辑，预览；
2. **保存文件**：快捷键：Ctrl + S 保存；
3. **预览**：保存后自动刷新预览；
4. **双击打开**：在 Windows 资源管理器里面关联 .md 文件后，双击 .md 文件，启动本程序，打开该 md 文件；
5. **双击打开注意**：本程序已经启动的情况下，在资源管理器双击 .md 文件，本程序自动打开双击的文件 -- 注意，如果有之前打开的 MD 文件没有保存，可能会丢失；
6. **文件拖放**：从资源管理器拖一个 .md 文件到本程序，本程序自动打开文件 -- 拖放到预览界面无效。可以拖放到编辑界面或者程序的其它地方；
7. **自动加载**：程序启动后自动加载上次打开的文件；
8. **新打开时自带加载上次的文件**：上次关闭本程序前的 .md 文件，新运行本程序时自动打开加载。

## 便捷功能
1. 常用文件夹：鼠标右键菜单
    -  **选择文件夹**：弹出文件夹选择窗口，选择一个文件夹，这个文件夹将会加入到这个常用文件夹列表；
    - **打开文件**：直接弹出选择的文件夹的文件打开窗口，直接在这个文件夹底下去打开文件，避免在 Windows 资源管理器里面去找寻要打开的文件夹；
    - **删除**：从这个常用文件夹列表里面把某一个文件夹去掉。

2. 最近打开的文件：最近打开的文件列表里面，显示的是最近打开过的文件。方便直接打开最近的文件。直接双击选中的文件打开文件；鼠标右键菜单：
    - **打开**菜单也能打开文件；
    - **删除**菜单是把这个文件的名字从列表里面删除，并不删除文件。
